---
# Gmail Integration

Provides email-based automation support.

---

# Supported Workflows

- OTP retrieval
- policy email parsing
- mailbox search
- message parsing

---

# Components

| File                  | Purpose
|-----------------------|------------------
| gmailClient.ts        | Gmail API client
| gmailService.ts       | service wrapper
| gmailOtpService.ts    | OTP extraction
| gmailPolicyService.ts | policy parsing
| gmailParser.ts        | email parsing
| gmailTypes.ts         | type definitions
